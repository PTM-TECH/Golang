## Go Introduction Program

# Overview

This is a simple Go (Golang) program that prints a personal introduction to the terminal. It demonstrates the basics of Go, including packages, imports, and the main function.

## Features

- Prints a name to the terminal.
- Shows a brief description of the developer.
- Demonstrates simple usage of the fmt package for output.

## Code Example

package main

import "fmt"

// Program entry point
func main() {
fmt.Println("Hello, I'm Patrick Mutua")
fmt.Println("A junior software engineer")
fmt.Println("I'm learning Golang")
}

## How to Run

Make sure Go is installed on your system.

Save the code in a file named main.go.

Open your terminal and navigate to the directory containing main.go.

Run the program with: - go run main.go

## Output

Hello, I'm Patrick Mutua
A junior software engineer
I'm learning Golang

## Purpose

This project is intended for beginners to get started with Go programming and understand:

    - The structure of a Go program.

    - How to use the fmt package to print output.

    - How to run Go programs from the terminal.
